# Lenskart Movies App - Flutter Assignment

## Overview
This is a Flutter application for the Lenskart Mobile Developer Internship.  
It allows users to:
- Browse movies from TMDB API
- Search movies by name
- Add movies to Favourites or Watchlist (unique per user)
- View movie details and play a movie (in-app notification)

---

## Features
- Splash Screen
- Bottom Navigation: Movies, Favourites, Watchlist
- Movie Search
- Circular rating indicator
- In-app notification ("Movie is Playing")
- Clean UI, loading & empty states
- State management using Provider

---

## Tech Stack
- Flutter & Dart
- Provider (State Management)
- TMDB API (HTTP)
- CachedNetworkImage (Image caching)
- Fluttertoast (Notifications)
